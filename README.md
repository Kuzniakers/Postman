# Postman Learning - Reqres.in Automation

This repository contains my Postman collection created as part of my learning process. It includes various API requests using the **Reqres.in** service for testing APIs.

## Collection Overview
The collection includes the following requests:
- ✅ **GET** - Fetch user details
- ✅ **POST** - Create a new user
- ✅ **PUT** - Update user details
- ✅ **DELETE** - Remove a user
- ✅ **POST** - Obtain an authentication token

## How to Use
1. Import the `.postman_collection.json` file into Postman.
2. Run the requests and observe the API responses.
3. Modify or extend the collection as needed.

## Technologies Used
- **Postman** for API testing

This project is purely for learning and testing API automation with Postman. 🚀
